## 🧠 How the AI Works

### Minimax Algorithm
The AI explores every possible future move
and picks the best one every time.

### Alpha-Beta Pruning
Skips unnecessary branches to make
the AI faster without losing accuracy.

### Difficulty Levels
| Level  | Description              |
|--------|--------------------------|
| Easy   | AI plays randomly        |
| Medium | AI thinks 2 moves ahead  |
| Hard   | AI is completely unbeatable |
